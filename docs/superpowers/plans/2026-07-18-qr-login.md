# 网易云二维码登录实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 增加默认二维码登录，同时保留手机号验证码登录。

**架构：** Rust 负责网易云二维码接口、二维码生成和会话落盘；现有 TypeScript 状态机负责模式切换、轮询生命周期和登录后复用 `AUTHENTICATED`。轮询只在设置抽屉打开且二维码模式有效时运行。

**技术栈：** Rust、Tauri 2、TypeScript、`qrcode` crate、现有 reducer 测试。
