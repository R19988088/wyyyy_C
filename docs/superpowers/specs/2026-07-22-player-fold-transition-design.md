# 播放器封面列表折叠过渡设计

## 目标

将大封面和封面列表之间的过渡替换为用户选定的方案 3：大封面向上折叠离开，封面列表从下方进入。

## 范围

- 仅修改 `_CoverMode` 内 `showingSwitchList` 所控制的封面/封面列表交接。
- 保留当前列表内容、玻璃层叠放、上下浏览、双击激活和大封面与歌曲列表的既有过渡。
- 打开时：封面以底边为轴向上折叠并淡出；列表从下方平移、淡入。
- 关闭时：同一动画自动反向执行。

## 验证

- 小部件测试断言打开列表时，封面折叠容器和列表切换容器均存在。
- 运行 `flutter test test/player_app_test.dart`、`dart analyze lib test` 与完整 `flutter test`。
